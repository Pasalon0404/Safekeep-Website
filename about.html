About SafeKeepVault

Built Because It
Should Exist

What if the hardware you already own is all you need?

Another Way to Think About It

Hardware wallets are great, and if you already have one, SafeKeepVault works right alongside it. But if you're just getting started, it's worth asking whether the $100–$200 you'd spend on a dedicated signing device might be better spent on Bitcoin itself — especially if you're not signing transactions every day.

Got a spare laptop sitting in a drawer? SafeKeepVault turns it into a fully capable, air-gapped signing machine at no extra cost. The hardware you already paid for becomes your vault. The money stays in Bitcoin.

The Software Is the Problem

Your laptop is fine — it's the operating system that's the issue. Windows and macOS are built to stay connected at all times, which is exactly what you don’t want near your Bitcoin keys. SafeKeepVault fixes this by swapping out the software at boot, not the hardware.

Plug in the thumb drive, restart, and the machine comes up offline with your existing files completely untouched. Pull it out when you're done and the computer is exactly as you left it — no trace of the session, nothing changed. Same screen, same keyboard, hardware you already own.

What We Believe

  • Self-custody shouldn't cost extra — the tools to hold your own Bitcoin securely should be free, open, and available to anyone willing to use them.
  • Trust should be earned, not assumed — every part of SafeKeepVault is open source. Read the code, verify the download, run it yourself.
  • You should see exactly what you're signing — open Bitcoin standards, human-readable output, every detail on screen before you commit.
  • The laptop in the drawer is good enough — it's a capable signing machine. You already paid for it.

Fully Open Source

MIT licensed. Full source on GitHub, every release signed and SHA-256 verified. If you find a bug, open an issue. If you want to contribute, open a pull request. Trust through transparency, not marketing.
